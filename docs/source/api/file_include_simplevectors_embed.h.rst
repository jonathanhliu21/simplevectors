Detailed Description
--------------------

A minimized version of vectors for embedded devices without access to the STL (such as on an Arduino, hence the ".h" instead of ".hpp" and the lack of a svector namespace). This file is meant to be a standalone file, so it is not included in :ref:`file_include_simplevectors_vectors.hpp`.
